# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| v1.x    | ✅ Active  |

## Reporting a Vulnerability

Do **not** file a public GitHub issue for security vulnerabilities.

Please email **security@autocyber.ai** with:

1. A description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Any suggested mitigations

We will respond within 48 hours and aim to release a fix within 7 days
for Critical/High severity issues.

## Responsible Disclosure

We follow coordinated disclosure. Please allow us reasonable time to
remediate before public disclosure.
